\Para{Differentialgeometrische Grundlagen}{}
In dieser Arbeit bezeichne $M$ immer eine $n$-dimensionale kompakte Mannigfaltigkeit
mit Rand, das heit einen Hausdorffraum, in dem jeder Punkt eine
Umgebung $U$ besitzt, die hom”omorph zu einer offenen Teilmenge $V$

$\phi \colon U \to V$ heit eine Karte von $M$. Ein System von Karten, deren 
Definitionsbereiche $M$ berdecken, heit ein Atlas von $M$.
Sind alle Kartenwechsel $\phi\circ\psi^{-1}$ stets
$\C^{k,\alpha}$-Diffeomorphismen,
so heit $M$ eine $\C^{k,\alpha}$-Mannigfaltigkeit.
Ein Punkt $p\in M$ mit $\phi(p)$ im Rand von $\RR^n_+$ heit Randpunkt,
sonst innerer Punkt von $M$. Der Rand $\partial M$ ist die Gesamtheit der 
Randpunkte,
das Innere $M^\circ$ die der inneren Punkte von $M$.
Eine Abbildung $f\colon M\to N$ zwischen Mannigfaltigkeiten heie vom Typ
$\C^{k,\alpha}$, falls fr alle Karten $\phi$ von $M$ und $\psi$ von $N$
die lokale Beschreibung $\psi \circ f \circ \phi^{-1}$ vom Typ 
$\C^{k,\alpha}$ ist.

Ein Vektorraumbndel heit normiert, falls auf jeder Faser eine

Im folgenden bezeichne $\T M$ immer das Tangentialbndel und $\T ^*M$

Eine $\C^k$-Abbildung $u\colon M\to \bi{E}$ heit ein $\C^k$-Schnitt in $\bi{E}$, falls
$u\circ \pi =\Id_M$. $\C^k(M,\bi{E})$ bezeichne die Menge aller $\C^k$-Schnitte in 
$\bi{E}$, es ist insbesiondere nicht die Menge aller $\C^k$-Abbildungen von 
$M$ in die Mannigfaltigkeit $\bi{E}$ gemeint. 
Schnitte in $\T M$ heien Vektorfelder auf $M$.
